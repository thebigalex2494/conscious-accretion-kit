# Minimal Example

The simplest possible setup: one `CLAUDE.md` and one rule file.

## Files

```
minimal/
  CLAUDE.md           -- Agent identity (15 lines)
  .claude/
    rules/
      quality-gates.md  -- Copy from framework/
```

## How to Use

1. Copy these files to any project directory
2. Open Claude Code in that directory
3. The orchestration protocol and quality gates are now active

This is the starting point. Add more rules, commands, and configuration as you need them.
